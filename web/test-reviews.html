Hardcover Reviews Widget Test Page

Configuration

Note: Enter a Hardcover username above and click "Update Widgets" to see their book reviews.

Method 1: Iframe Embedding

The simplest way to embed reviews. Just add an iframe to your HTML:

<iframe src="${apiUrl}/static/reviews-embed.html?username=gouthamve" width="100%" height="600" frameborder="0"> </iframe>

Method 2: JavaScript Widget

For more control and customization, use the JavaScript widget:

<div data-hardcover-review-widget data-username="gouthamve" data-api-url="${apiUrl}" data-max-width="800px" data-show-powered-by="true"> </div> <script src="${apiUrl}/static/review-widget.js"></script>

Advanced Widget Configuration

The JavaScript widget supports additional configuration options:

// Initialize programmatically with custom options new HardcoverReviewWidget(element, { apiUrl: '${apiUrl}', username: 'gouthamve', maxWidth: '600px', maxReviewLength: 200, // Truncate reviews after 200 characters showPoweredBy: false // Hide "Powered by Hardcover" link });
