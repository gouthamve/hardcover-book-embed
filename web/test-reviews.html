Book Reviews Widget Test Page

This page demonstrates how to embed the Hardcover book reviews widget on your website.

Live Example

How to Embed

To embed the book reviews widget on your website, use the following iframe code:

<iframe src="http://localhost:8080/static/reviews-embed.html?username=YOUR_USERNAME" width="100%" height="600" frameborder="0"> </iframe>

Replace YOUR_USERNAME with your Hardcover username.

Customization Options

  • Width: Set to any value (e.g., "800px", "100%")
  • Height: Adjust based on your layout needs

API Endpoint

The widget fetches data from:

GET /api/books/reviews/{username}

Response format:

{ "books": [ { "book": { "id": 123, "title": "Book Title", "slug": "book-slug", "image": { "url": "https://..." } }, "rating": 4.5, "review_raw": "This is the review text...", "review_html": "<p>This is the review text...</p>", "reviewed_at": "2024-01-15T10:30:00Z", "review_has_spoilers": false, "url": "https://hardcover.app/books/..." } ], "count": 10, "updated_at": "2024-01-15T12:00:00Z" }
